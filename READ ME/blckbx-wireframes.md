## Key Interaction Flows

### Flow 1: Create Multi-Destination Itinerary
```
1. User clicks "Create New" â†’ selects "Itinerary"
2. Enters basic info (name, assistant) â†’ creates with 1 default destination
3. User edits destination name/dates
4. User clicks "Add Destination" â†’ modal opens

6. Destination bar now shows: [Lisbon] â†’ [Algarve]
7. User clicks "Edit Travel Between" â†’ Inter-travel modal
8. Selects travel type, enters details â†’ saves
9. User continues adding items to each destination
```

### Flow 2: Auto-Fill Item
```
1. User is adding accommodation
2. Pastes Booking.com URL into Quick Fill input
3. Clicks "Auto-fill" â†’ loading state
4. n8n webhook extracts data â†’ returns JSON
5. Form fields populate automatically
6. User reviews, edits if needed
7. Clicks "Save" â†’ item created with sourceType: "autofill"
```

### Flow 3: Convert List to Itinerary
```
1. User has a list with 10 hotels, 5 restaurants

4. User selects 2 hotels, 3 restaurants

6. Clicks "Create Itinerary"
7. New itinerary created with selected items copied over
8. Original list remains (or deleted if user chose)
```
