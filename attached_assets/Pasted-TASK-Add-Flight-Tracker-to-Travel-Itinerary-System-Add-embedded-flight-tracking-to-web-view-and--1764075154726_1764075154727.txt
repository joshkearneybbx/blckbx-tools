TASK: Add Flight Tracker to Travel Itinerary System

Add embedded flight tracking to web view and clickable links to PDF for outbound and return flights.

IMPLEMENTATION:

1. CREATE FILE: client/src/components/FlightTracker.tsx
```typescript
interface FlightTrackerProps {
  flightNumber: string;
  width?: string;
  height?: string;
}

export function FlightTracker({ 
  flightNumber, 
  width = '100%', 
  height = '500px' 
}: FlightTrackerProps) {
  const cleanFlightNumber = flightNumber.replace(/\s/g, '').toUpperCase();
  
  return (
    <div className="flight-tracker-container my-6">
      <h3 className="text-lg font-semibold mb-3">Track Your Flight</h3>
      <iframe 
        src={`https://www.flightradar24.com/data/flights/${cleanFlightNumber.toLowerCase()}`}
        width={width}
        height={height}
        style={{ border: 'none', borderRadius: '8px', boxShadow: '0 2px 8px rgba(0,0,0,0.1)' }}
        title={`Flight ${cleanFlightNumber} Tracker`}
        loading="lazy"
      />
      <p className="text-sm text-gray-600 mt-2">
        Live flight tracking powered by FlightRadar24
      </p>
    </div>
  );
}
```

3. UPDATE: PDF Template (find your ItineraryPDFTemplate file)

Add this helper function at the top of the component:
```typescript
const getFlightTrackerUrl = (flightNumber: string) => {
  const clean = flightNumber.replace(/\s/g, '').toLowerCase();
  return `https://www.flightradar24.com/data/flights/${clean}`;
};
```

In the Outbound Travel section, add after the travel info fields:
```typescript
{itinerary.outboundTravel?.flightNumber && (
  <Link 
    src={getFlightTrackerUrl(itinerary.outboundTravel.flightNumber)}
    style={styles.flightTrackerLink}
  >
    ✈️ Track Flight {itinerary.outboundTravel.flightNumber}
  </Link>
)}
```

In the Return Travel section, add after the travel info fields:
