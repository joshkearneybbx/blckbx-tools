## FIX 3: Add Notes Field to Activities/Restaurants/Accommodations

Add a "notes" field for custom information like "Tickets only available in person" or "Arrive 30 mins early".

### Database Changes:
Add "notes" TEXT column to:
- activities table
- dining table
- accommodations table

### Form Changes (Wizard):
In the activity/dining/accommodation forms, add a textarea:

<div>
  <label>Notes (optional)</label>
  <textarea
    placeholder="e.g., Arrive 30 mins early, tickets only at door, dress code required, etc."
    rows="3"
    value={formData.notes}
    onChange={(e) => setFormData({...formData, notes: e.target.value})}
  />
</div>

### Display Changes:
In both web view and PDF, show notes if they exist:

{item.notes && (
  <View style={styles.notesSection}>
    <Text style={styles.notesLabel}>Notes:</Text>
    <Text style={styles.notesText}>{item.notes}</Text>
  </View>
)}
